> A secure login web application with real bcrypt password hashing, input validation, session management, and two-factor authentication. No backend required — runs entirely in the browser.

---

## ✨ Features

### 🔐 Security
- **Real bcrypt hashing** using `bcryptjs` (cost factor 10 = 1024 rounds)
- **Unique salt** generated per user automatically by bcrypt
- **Plain-text passwords never stored** — only hash is saved
- **Two-Factor Authentication** (6-digit TOTP simulation)
- **Input validation** — username format, email format, password strength, match check

### 🖥️ UI / UX
- Register & Login tabs with smooth switching
- Password strength meter (5-level colour bar)
- Show/hide password toggle
- Error and success alert messages
- Demo account pre-loaded for instant testing

### 📊 Dashboard (after login)
- Displays full account details
- Security status checklist
- **Shows your actual bcrypt hash** so you can see what gets stored
- bcrypt format explanation: `$2a$10$[salt][hash]`

### 📄 Pages
- **Login** — main authentication tool
- **About** — explains bcrypt, salting, 2FA, and validation
- **Contact** — contact form with info panel

## 🔑 How bcrypt Works

```
Plain password:  "Demo@1234"
         ↓
bcrypt(password, saltRounds=10)
         ↓
Hash: "$2a$10$Kd3vTg...8uYz" ← stored in DB
         ↓
Login: bcrypt.compare(input, storedHash) → true/false
```

- **Same password → different hash every time** (due to unique salt)
- **Cannot reverse** the hash to get the original password
- **10 rounds** = 1024 iterations — slow enough to defeat brute force
